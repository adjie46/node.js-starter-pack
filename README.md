# node.js-starter-pack
node.js-starter-pack

Node.js Starter Pack With Migration Database dan Seeder Data

Installasi

- npm i --save
- npm install nodemon
- npx sequelize db:migrate
- npx sequelize-cli db:seed:all
- npm run server
